Uma outra bela demonstração de CSS Externo

Note que nesse caso não existe nenhum código CSS na página HTML, pois o arquivo que contém as formações é externo. Para isso você deve criar um arquivo .css em uma pasta "Style" e indicá-lo através de um link dentro HEAD :).

Neste caso todos os parágrafos irão herdar as características definidas pelo arquivo .css, note que qualquer arquivo HTML qur você criar e indicar o arquivo css externo será formato pelo padrão definido, isso é muito útil para projetos com mais de um arquivo HTML.
